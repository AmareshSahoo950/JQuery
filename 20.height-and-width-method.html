Lecture - 22

All the height() and width() related methods

Definition

width() -> get/set the width of the content. 0

innerWidth() -> get/set the width of the content + padding. 0

outerWidth() -> get/set the width of the content + padding + border-width. 0

outerWidth(true) -> returns the width of the content + padding + border-width + margin. 0

height() -> get/set the height of the content. 0

innerHeight() -> get/set the height of the content + padding. 0

outerHeight() -> get/set the height of the content + padding + border-width. 0

outerHeight(true) -> returns the height of the content + padding + border-width + margin. 0
